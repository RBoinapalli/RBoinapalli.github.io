CSCE 242: Web Applications

Rithika Boinapalli

CSCE 242: Web Applications introduces students to the core principles of developing dynamic and responsive websites. The course
covers front-end technologies like HTML, CSS, and JavaScript, as well as server-side programming and database integration. By the
end, students will be equipped with the skills to design, build, and deploy modern web applications.
