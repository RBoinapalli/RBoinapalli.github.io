CSCE 242: Web Applications

Rithika Boinapalli

This course introduces students to the important concepts in developing a dynamic and responsive website. It covers front-end technologies such as HTML, CSS, and JavaScript, besides server-side programming and database integration. By the end, students will have acquired skills in designing, building, and deploying modern web applications.

Assignments

  1. Basic HTML
  2. Basic CSS
