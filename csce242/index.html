CSCE 242: Web Applications

Assignments

  • Assignment 1 - Basic HTML

    • For Assignment 1, a new web page
    was created on a classroom-
    appropriate topic, starting from
    scratch. The page includes
    properly formatted HTML with a
    doctype, title tag, h1 heading,
    two paragraphs, two images, a list,
    a table, and external links, all
    accompanied by comments indicating
    the user's name and course.

  • Assignment 2 - Basic CSS

    • For Assignment 2, a new web page
    was created with an index.html page
    that includes a title, navigation links,
    paragraphs, images, and a table.
    The main portal was enhanced by
    changing the font, adjusting the
    color scheme, adding a background,
    formatting links, and ensuring the
    design is visually appealing with
    matching colors and styles.

  • Assignment 3 - Page Layout

    • For Assignment 3, a single webpage
    was created using HTML5 elements
    such as header, footer, nav, section,
    aside, and article, structured as a
    traditional HTML page with a header,
    content, and footer. The page
    utilized Flexbox for layout, including
    a navigation bar that adjusts
    between horizontal and vertical
    orientations based on screen size,
    with added background colors,
    appropriate padding, and styling to
    ensure the webpage is visually
    appealing and matches the layout
    in the provided diagram.
